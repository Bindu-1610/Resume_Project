# Resume_Project
Smart Resume Analyzer

A Machine Learning based web application that analyzes resumes and predicts job category.

##  Features
- Upload Resume (PDF or Text)
- Automatic Category Prediction
- Built using Flask and Scikit-learn
- Clean Web Interface

##  Technologies Used
- Python
- Flask
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression

## 📂 Dataset
Resume Dataset (Kaggle)

## 🛠 How to Run

### Step 1: Install Dependencies
pip install -r requirements.txt

### Step 2: Train Model
python train_model.py

### Step 3: Run Application
python app.py

Open browser:
http://127.0.0.1:5000/

## 📊 Algorithm Used
- TF-IDF
- Logistic Regression
